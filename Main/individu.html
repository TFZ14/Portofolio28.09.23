Web Personality Gather

[UI/UX] Web Personality Gather was created to house classifications of human traits based on scientific, pseudoscientific, and belief-based systems. The purpose of this website is to provide users with easy access to information regarding human trait classification and their interval. Currently, it encompasses human traits classified according to Zodiac signs, MBTI, Enneagram, Tritype, and Chinese zodiac signs (Shio).

Personality Gather Mobile

[UI/UX] Personality Gather application was developed with the aim of enabling users to input their personality types based on zodiac signs, MBTI, Enneagram, and Tritype classifications. The primary concept behind creating this application was to record an individual's (user's) type after they complete a personality test provided through a given link. As it progresses, this application can be enhanced with a database and transformed into a discussion forum for users engaged in the field of psychology, ranging from amateurs to experts, covering both scientific-based approaches and belief-based perspectives.
